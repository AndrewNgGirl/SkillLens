# Marketing Assets

Standalone, public-facing marketing material that is **not** referenced by the
SkillLens code. These files exist only to be opened directly in a browser or
shared as-is.

| File | Purpose |
| --- | --- |
| `skilllens-promo-slides.html` | Single-file promo slide deck. Open in any browser; no build step required. |

If you add new marketing files here, keep them self-contained (zero-dependency
HTML, image bundles) so they can be shared without running the web app.
